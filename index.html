Help & Documentation

👋 Quick Start

Range Navigator is an interactive poker chart viewer for iPad. Tap positions on the table to build paths and view corresponding range charts.

Use the format buttons to switch between 6-max and Heads-Up, and the pack dropdown to select different strategy files.

📋 Image Naming Convention

When adding custom chart images, follow this naming system. Charts are organized by situation — who's acting and what action they're taking.

Situation Examples File Name Pattern

BTN.png		 Position.png

BTN_3B-CO-7bb.png		 Actor_3B-Opener(-size).png

SB_CC-BTN.png		 Caller_CC-Opener(-size).png
vs 3B
(we opened, opponent 3bet)		 UTG-CO_3B.png
UTG-CO_3B-22bb.png		 Actor-Opener_3B(-size).png

UTG_4B-BTN-22bb.png		 Actor_4B-Opponent(-size).png
vs 4B
(in a 3bet pot, opponent 4bets)		 BTN-UTG_4B.png
BTN-UTG_4B-23bb.png		 Actor-Opponent_4B(-size).png

(cold 4bet, 3 players)		 BB_C4B-SB-BTN.png

(we squeeze, 2+ callers)		 BB_Squeeze-BTN-UTG.png

Naming Rules:

  • _ separates the actor position from their action (e.g., BTN_3B)
  • - separates different players in the situation (e.g., CO-BTN)
  • Size goes at the very end (e.g., -7bb, -22bb)
  • Create size variants by adding -sizeName: BTN_3B-CO.png (base) + BTN_3B-CO-7bb.png + BTN_3B-CO-9bb.png → tabs appear automatically

File Placement: Charts live in position subfolders: images/{pack}/{position}/filename.png

⚙️ Settings & Features

Pack Selector — Choose which strategy pack to view. Different packs contain strategies for different stack depths, rake structures, or game variants.

Format Switcher — Switch between 6-max and Heads-Up table layouts.

Auto-Reset Toggle — Automatically clears your path when you tap the felt outside of positions. Handy for rapid lookups.

Show Sequence Toggle — Display the next action in the sequence alongside your current range. Helpful for understanding flow.

Overview Manager (⋯ button) — Tap the three-dot menu on any position to:

  • Add custom overview charts (e.g., combined RFI ranges for a position)
  • Set a default overview (opened when double-tapping a position)
  • Drag to reorder your overview list
  • Edit labels and file paths inline

Frequency Display (% button) — In the chart modal, tap the % button to toggle frequency percentages on/off.

Custom Overviews — All overviews you add are saved locally in your browser. They sync across devices only if you back them up via data.js.

➕ Adding Your Own Charts

You can add new chart images to the app using two methods:

Method A: In-App (easiest)

  • Tap the ⋯ button on any position → Manage
  • Tap "Add new entry"
  • Fill in the position, action, label, and file path (relative to images/)
  • Create or place your PNG file at that path, then reload

Method B: Edit data.js

  • Create a chart image and name it using the convention above
  • Place it in the correct folder: images/{pack}/{position}/{filename}.png
  • Add a node entry to the pack's nodes array in data.js (copy an existing node as a template)
  • Double-click build-manifest.ps1 to rebuild the file index
  • Reload the app to see your new chart

💡 Tips & Tricks

  • Double-tap a position to open its default overview
  • Drag in the chart view to pan when zoomed in
  • Pinch or scroll to zoom in/out of charts
  • Tap outside the chart to close the modal (or use the ✕ button)
  • Drag overview items in the Manager to reorder them
  • All your custom overviews are saved locally — no internet needed
